VASP+Wannier90--An Introduction

Posted on 2017-08-06   |   次阅读   |   visitors

Compile

  1. wannier90-1.2 compile
    Go to http://www.wannier.org/download.html and download the wannier90-1.2 version, not the newest version.
    Modify ‘make.sys’ file according to your system, and then
    make all

  2. Recompile VASP
    Modify the following parts:

    LIB = -L../vasp.5.lib -ldmy \

    ../vasp.5.lib/linpack_double.o \
../wannier90-1.2/libwannier.a $(SCA) (LAPACK)(LAPACK)(BLAS)

    CPP = $(CPP_) -DMPI -DHOST=\”LinuxIFC\” -DIFC \

    -DCACHE_SIZE=4000 -DPGF90 -Davoidalloc  \
-DMPI_BLOCK=8000 -Duse_collective -DscaLAPACK -DNGZhalf <span style="color:red">-DVASP2WANNIER90</span>

Then 

make vasp
