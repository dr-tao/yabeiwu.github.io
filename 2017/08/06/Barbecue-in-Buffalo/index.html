Barbecue in Buffalo： Fort Niagara State Park.

Posted on 2017-08-06   |   次阅读   |   visitors

  1. 行程前准备

    1. 召集同学。大家都很久没有出去烧烤了，问了一下就召集了十几个人。
    2. 购买食材。根据人数去Bjs，沃尔玛购买一定量的食材，为了省事又省钱，我们购买了大量的鸡翅，在中国超市买了腌制鸡翅的调料包。另外购买了牛仔骨，羊肉，香菇，金针菇，培根，生菜等，大量饮料。
    3. 地点选择。我之前去过Fort Niagara State Park，觉得这个公园很不错，毗邻安大略湖边，景色优美，离我们的地方最近的一个景点，所以就选择了这里。
    4. 食材的前期处理。我们把鸡翅分成两拨，一波煮半熟，一波煮全熟，用买好的料包进行腌制。香菇需要泡好，其他的没有进行处理。
    5. 车辆安排。我们一行12人，正好三辆车，公园要收8刀没票，必须最优化。

  2. 烧烤

    1. 公园里面有烤架，自带锡纸或者烧烤垫。把购买的食材都放到公园桌子上。烤架铺上锡纸，然后放碳和charcoal lighter，可以直接引燃，等火灭了就可以放置食材。
    2. 根据口味选择烤肉的程度。

不完美事项：天公不作美，预报天气是多云，没想到刚开始烧烤下起来暴雨，不过比较好的事情是，只下了半个小时，然后就放晴了，还算不完美中的幸运。
