How to install phonopy with/without root access?

Posted on 2017-08-06   |   次阅读   |   visitors

This will has two situations: 1, root account; 2, without root access. So we will discuss all the situations following:

1, root account:

if you are a root account, everything becomes much easier for you. All you just need to do is the following:
install python development packages:

numpy, numpy-dev, matplotlib, python-lxml, python-yaml

if you are using a Ubuntu/Debian system, you just need to：

sudo aptitude install python-dev python-numpy 
\python-matplotlib python-tk python-lxml python-yaml

And after that, go to the phonopy directory, and using command:

sudo python setup.py install

Then edit your ~/.bashrc file, such as:

export PATH=~/opt/phonopy-0.6.1:$PATH

2, without root access:

Things become much more complicated, but don’t worry, we can handle this.
1, you should install your local python program.
install local python without root access (see the following, copy from this link)
choose your own python version and your installed directory.

wget https://www.python.org/ftp/python/2.7.5/Python-2.7.5.tgz
tar -xzf Python-2.7.5.tgz
cd Python-2.7.5
mkdir -p /home/liudiwei/software/python27 
./configure --prefix="/home/liudiwei/software/python27"
make
make install

2, install setuptools
setuptools is prepared for pip program, just using following command:

wget https://www.python.org/ftp/python/2.7.5/Python-2.7.5.tgz
tar -xzf Python-2.7.5.tgz
cd Python-2.7.5
mkdir -p /home/liudiwei/software/python27 
./configure --prefix="/home/liudiwei/software/python27"
make
make install

3, install pip
keep remembering that you should use your own python program.

wget https://www.python.org/ftp/python/2.7.5/Python-2.7.5.tgz
tar -xzf Python-2.7.5.tgz
cd Python-2.7.5
mkdir -p /home/liudiwei/software/python27 
./configure --prefix="/home/liudiwei/software/python27"
make
make install

4, install related library
first goes to /your/path/to/python/bin directory

cd /home/liudiwei/software/python27/bin/
./pip install simplejson
./pip install redis
./pip install numpy
./pip install scipy
./pip install sklearn
./pip install matplotlib

Because your system already has a python program, so you can rename your python to a different one, by changing the ~/.bashrc file

alias mypython='/home/liudiwei/software/python27/bin/python'

5, install phonopy
using your local python to install the phonopy, and don’t specify anything.

mypython setup.py install

After all the installation, have fun in using phonopy.
